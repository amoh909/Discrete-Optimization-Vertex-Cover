- Random graphs G(n, p) with:
  - n ∈ {10, 16, 24, 32}
  - p ∈ {0.15, 0.30, 0.50, 0.80, 0.90}
  - 3 trials per configuration

- Grid graphs:
  - 3×3, 4×4, 5×5, 4×6

- Bipartite graphs:
  - (n1, n2, p) = (6,6,0.3), (8,8,0.4), (10,12,0.3), (12,12,0.5)
  - 2 trials per configuration

- Cycle graphs:
  - n ∈ {5, 6, 7, 8, 9, 10, 11, 15}

- Complete graphs:
  - n ∈ {4, 6, 8, 10, 12, 16, 20, 24}

- Near-cliques:
  - n ∈ {12, 16, 20}
  - edge removal fractions ∈ {0.10, 0.15, 0.20}
